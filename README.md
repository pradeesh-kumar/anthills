# Anthills
![logo.png](logo.png)

Anthills is a distributed worker framework written in Java

## Features
* Work Request Processor
* Scheduler -> Leasable and Independent
* Concurrency
* Idempotency
* Rest API and UI based summary


## Supports
Scheduled Worker
Runs on every specified schedule

Leased Scheduler Worker
WorkRequest Worker
Leased WorkRequest Worker
